# Issuance Deployment Design

This directory documents deployment of the issuance system components
(`RewardsEligibilityOracle`, `IssuanceAllocator`, and related experimental
targets such as `PilotAllocation`).

For now, the most complete design reference still lives in
`incoming/issuance/Design.md`. That document is treated as the canonical
design while we migrate and trim documentation into this directory.

- Canonical design (single source of truth, to be moved here):
  `incoming/issuance/Design.md`

All other deployment docs in this directory should be consistent with that
design. Once the migration is complete, the full design document will live
here and `incoming/issuance/Design.md` will be treated as
archive/background material.
